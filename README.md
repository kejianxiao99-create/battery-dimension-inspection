
# Battery Dimension Inspection

## Project Objective

This project aims to build a vision-based battery dimension inspection system using Python and OpenCV.

The final goal is to:

- Detect batteries automatically
- Measure battery dimensions
- Convert pixel measurements into millimetres
- Evaluate measurement accuracy
- Build an engineering-level inspection workflow
